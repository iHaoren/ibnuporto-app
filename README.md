A modern, responsive digital profile website built with React and Vite. This project serves as a comprehensive showcase of my professional background, skills, and learning journey in technology.

## Purpose

This website was created with three main goals:

1.  **Recruiter Engagement:** To provide a professional and accessible platform for recruiters to learn about my qualifications and experience.
2.  **Connection:** To make it easier for industry professionals and collaborators to connect with me.
3.  **Technological Growth:** As a hands-on project to deepen my understanding of modern web development technologies and best practices.

## Key features

- **Interactive Gallery:** A visual showcase of projects and personal highlights.
- **Documentation & Certificates:** Organized display of training certificates and professional documentation.
- **Responsive Design:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
- **Modern UI/UX:** Built with smooth animations (AOS, Animate.css) and a clean aesthetic using Sora typography.
- **Dynamic Typewriter Effects:** Engaging introductory animations using `react-type-animation`.

## Built with

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons:** [Remix Icon](https://remixicon.com/)
- **Animations:** [AOS](https://michalsnik.github.io/aos/) & [Animate.css](https://animate.style/)
- **Deployment:** GitHub Pages
